# Indexer Client

## Getting Started

### Installation

<Tabs items={["TypeScript", "Python"]}>
<Tab>
```typescript copy
pnpm install @dydxprotocol/v4-client
```
</Tab>
<Tab>
```python copy
pip install v4-client-py
```
</Tab>
</Tabs>

### Initializing the Client

<Tabs items={["TypeScript", "Python"]}>
<Tab>
```typescript copy
import { IndexerClient } from "@dydxprotocol/v4-client";

const client = new IndexerClient(Network.testnet().indexerConfig);
```
</Tab>
<Tab>
```python copy
from v4_client_py import IndexerClient

client = IndexerClient(
    config=Network.testnet().indexer_config,
)
```
</Tab>
</Tabs>
