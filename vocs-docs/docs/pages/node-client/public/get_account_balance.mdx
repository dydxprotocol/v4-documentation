### Get Account Balance

The [Get Account Balance] query retrieves the balance of a specified account address for a particular denomination.

This function is used to obtain the current funds of a token available in a wallet within the blockchain network.

[Get Account Balance]: https://github.com/cosmos/cosmos-sdk/tree/main/x/bank#balance

#### Method Declaration

:::code-group

async def get_account_balance(

#### Parameters

| Parameter | Location  | Type      | Required  | Description                                            |
| --------- | --------- | --------- | --------- | ------------------------------------------------------ |
| `address` | query     | [Address] | true      | The wallet address that owns the account.              |
| `denom`   | query     | [Denom]   | true      | Denomination of the token associated with the request. |

#### Response

| Status | Meaning     | Schema     |
| ------ | ----------- | ---------- |
