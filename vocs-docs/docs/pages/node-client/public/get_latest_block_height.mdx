### Get Latest Block Height

Retrieves the most recent block height from a blockchain network.
Internally it uses the [`Get Latest Block`](#get-latest-block) API call.

#### Method Declaration

:::code-group

```python [Python]
async def latest_block_height(self) -> int
```

```typescript [TypeScript]
async latestBlockHeight(): Promise<number>
```

```rust [Rust]
pub async fn get_latest_block_height(&mut self) -> Result<Height, Error>
```

:::

#### Parameters

#### Response

| Status | Meaning       | Schema   | Description                           |
| ------ | ------------- | -------- | ------------------------------------- |
| `200`  | [OK]          | [Height] | The latest block height.              |
| `400`  | [Bad Request] |          | The request was malformed or invalid. |
