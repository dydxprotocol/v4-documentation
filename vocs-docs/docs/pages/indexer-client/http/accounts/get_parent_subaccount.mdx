### Get Parent Subaccount

Query for the parent subaccount, its child subaccounts, equity, collateral and margin. See more information on parent subaccounts [here](/concepts/trading/isolated-positions#parent-subaccounts).
e.g. parent subaccount 0 has child subaccounts 128, 256,...

| Parameter | Location  | Type                  | Required  | Description                                                           |
